# uhadni_cislo_java
Jednoducha hra pre hadanie cisla napisana v Java
******************************************************************************************
Jednoducha hra pre hladanie cisla napisana v Java 17.0.3 (raspberry pi4). V hre je ulohou hraca uhadnut cislo ktore je nahodne generovane prostrednictvom kodu. Hrac zada rozsah v ktorom je cislo generovane a nasledne sa urci kolko pokusov ma hrac na to aby spravne uhadol cislo. Pocet pokusov musi byt vzdy maximalne 1/2 z maximalneho rozsahu. Nasledne ked hrac hada tak kod vypisuje ci hrac uhadol spravne cislo a ak tak vypise spravne cislo. Pokial hrac neuhadne spravne kod vypise pocet ostavajucich pokusov pre uhadnutie. Ked hrac uhadne hra sa spita ci chce hrac hrat znova.
